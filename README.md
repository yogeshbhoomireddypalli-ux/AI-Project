# 📈 Linear Regression from Scratch (Gradient Descent)

This project implements **Linear Regression using Gradient Descent from scratch in Python**, without using external libraries like NumPy, pandas, or scikit-learn.

It also includes basic regression evaluation metrics.

---

## 📌 Project Overview

This project demonstrates:

- ✅ Min-Max Normalization
- ✅ Linear Regression Model
- ✅ Gradient Descent Optimization
- ✅ Loss Tracking
- ✅ Model Evaluation Metrics:
  - Mean Squared Error (MSE)
  - Root Mean Squared Error (RMSE)
  - R² Score
